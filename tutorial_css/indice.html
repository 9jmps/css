Display y posicion


Display

Display es la propiedad mas importante para controlar las estructuras. Cada elemento tiene un valor por defecto para la mayoria de los elementos es usualmente bloc (de bloque) o inline(en linea). Un elemento que es block es comunmente llamado block-level.


inline

el span es el elemento inline estandar. un element inline puede contener algo de texto dentro de un parrafo sin interrumpir el flujo del parrafo.

Posiscionamiento

La propiedad position de CSS, especifica como un elemento es posicionado en el documento, como un elemento es posicionado en el documento. las propiedades top,right,bottom y left determinan la ubicacion final de los elementos posicionados.
Tipos de posicionamiento Un Elemento posicionado: Es un elemento cuyo valor computado de position es relative,absolute,fixed.

  • Elemento Estatico: El elemento es posicionado de acuerdo al flujo normal del documento.Las propiedades top,right,bottom,leftl y z-index, no tiene efecto.Es el valor dado por defecto.
  • Elemento Relativo: El elemento es podicinado de acuerdo al fujo normal del documento, y luego es desplazado con relacion a si mismo, con base en los valores de top, right, bottom y left. El dezplazamiento no afecta la posicion de ningun otro elemento. position:relative.
  • Elemento Absoluto: el elemento es removido del flujo normaldel docuemnto, sin crearse espacio alguno para el elementoen el esquema de la pagina. es posicionado relativo a su ancestro posicionado mas cercano. su posicion final esta determinada por los valore de top,right,bottm y left. position:absolute.
  • Elemento Fijo: El elemento es removido del flujo normal del documento,sin crearse espaci alguno para el elemento de la pagina. Es posicionado con relacion al bloque contenedos. position:fixed.

z-index

La propiedad de css z-index indica el orden de un elemento y sus desendientes.Cuando varios elementos se superponen, z-order determina cuales cubren a otros.
Los valores que maneja son:

  • auto:la caja no estalece un nuevo contexto de apilamiento
  • integerEste entero es el nivel de apilamiento de la caja generada en el actual contexto de apilamiento.El nivel de apilamiento es 0.
